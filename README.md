# Open in CMD

Based on [open-conemu-here](https://github.com/ziyasal/atom-open-conemu-here), "atom_open_in_cmd" package allows user to open a CMD in atom from tree-view & workspace & editor.

To install;
```shell
apm install open_in_cmd
```
